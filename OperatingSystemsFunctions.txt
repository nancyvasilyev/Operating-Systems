Memory Management: is an operating system function which handles the primary memory. Memory management keeps track of all the locations regardless of its allocation.  This function ensures accuracy and availability of data that is being added or removed.
Processor Management: is an operating system function that manages the time and access each process has to the processor. This function has access to keep track of processes status which is also known as traffic controller. When a process is no longer required this function has the ability to remove specific processors.
File Management: this function allows for easy navigation among files that are stored on a device. File management keeps track of where all the information is stored. When this information is needed it is then retrieved with easy access and navigation.
